Getting Started

This guide will help you get up and running with the BOOSE interpreter.

Prerequisites

  • .NET 8.0 SDK or later
  • Visual Studio 2022 or Visual Studio Code (with C# extension)
  • Windows operating system (for Windows Forms UI)

Installation

Building from Source

  1. Clone the repository:

    git clone <repository-url>
cd ase-assignment-davidcralph

  2. Restore dependencies:

    dotnet restore

  3. Build the project:

    dotnet build

  4. Run the interpreter:

    dotnet run --project BOOSE.Interpreter

Running Tests

To verify your installation, run the unit tests:

dotnet test

Your First BOOSE Program

Let's create a simple BOOSE program that draws a smiley face.

Using the GUI

  1. Launch the BOOSE Interpreter application
  2. Type the following code in the program editor:
pen 255,255,0
moveto 200, 200
circle 100

pen 0,0,0
moveto 170, 170
circle 10

moveto 230, 170
circle 10

moveto 150, 230
drawto 250, 230
  1. Click the "Run" button
  2. View the output in the canvas area - you should see a yellow smiley face!

Understanding the Code

  • pen 255,255,0 - Sets the pen color to yellow (Red=255, Green=255, Blue=0)
  • moveto 200, 200 - Moves the pen to position (200, 200) without drawing
  • circle 100 - Draws a circle with radius 100 at the current position
  • pen 0,0,0 - Changes pen color to black
  • drawto 250, 230 - Draws a line from current position to (250, 230)

Loading Example Programs

The project includes numerous example programs in the BOOSEexamplePrograms folder:

  1. Click File > Open in the BOOSE Interpreter
  2. Navigate to BOOSEexamplePrograms/1BasicDrawing/
  3. Select 1unrestrictedDrawing.boose
  4. Click Run to execute the program

Example Program Categories

  • 1BasicDrawing: Simple drawing commands (moveto, drawto, circle, rect, pen)
  • 2Variables: Integer, real, array, and boolean variables
  • 3IfsAndLoops: If/else statements, while loops, and for loops
  • 4Methods: Method definitions and calls
  • 5miscellaneous: Larger programs demonstrating multiple features

Basic BOOSE Syntax

Drawing Commands

moveto x, y          # Move pen to position (x,y)
drawto x, y          # Draw line to position (x,y)
circle radius        # Draw circle at current position
rect width, height   # Draw rectangle at current position
pen r, g, b          # Set pen color (RGB values 0-255)
write "text"         # Write text at current position
reset                # Reset pen to origin (0,0)

Variables

int myNumber = 42
real price = 19.99
int result
result = myNumber * 2

Arrays

array int numbers 10     # Declare array of 10 integers
poke numbers 0 = 100     # Set element at index 0 to 100
peek value = numbers 0   # Read element at index 0 into variable

Control Flow

if x > 10
    circle 50
else
    circle 25
end if

while count < 5
    circle count * 10
    count = count + 1
end while

for i = 1 to 10 step 2
    moveto i * 20, 100
    circle 10
end for

Methods

method int square int num
    square = num * num
end method

call square 5
write square

Debugging

Using the Debug Window

  1. Click View > Debug to open the debug window
  2. Run your program - execution traces will appear in the debug window
  3. Review output to identify syntax errors or runtime issues

Common Errors

Syntax Error: Unexpected token

  • Check your command spelling and syntax
  • Ensure proper spacing and commas in parameters
  • Verify all control structures have matching end statements

Variable not declared

  • Declare variables before using them: int myVar = 0
  • Check for typos in variable names

Index out of bounds

  • Array indices must be within the declared size
  • Arrays are 0-indexed (first element is at index 0)

Programmatic Usage

You can also use the BOOSE interpreter programmatically in your C# code:

using BOOSE.Interpreter;

// Create a canvas
BOOSECanvas canvas = new();
canvas.Set(400, 400);

// Create program runner
BOOSEProgramRunner runner = new(canvas);

// Execute BOOSE code
string program = @"
    pen 255,0,0
    moveto 100,100
    circle 50
";

try
{
    runner.ExecuteProgram(program);
    // Canvas now contains the drawing
}
catch (Exception ex)
{
    Console.WriteLine($"Error: {ex.Message}");
}
