Provides a Windows Forms-based implementation of the BOOSE Canvas for graphical drawing operations. Implements the abstract Canvas class with GDI+ rendering capabilities.

Reset()

Resets the canvas state to its initial configuration.

public override void Reset()

Remarks

This method resets the cursor position to (0, 0) and sets the drawing color to the theme default. It does not clear the canvas content.

Set(int, int)

Initializes or reinitializes the canvas with the specified dimensions.

public override void Set(int width, int height)

Parameters

width int

The width of the canvas in pixels.

height int

The height of the canvas in pixels.

Remarks

This method disposes of any existing graphics resources and creates a new bitmap with anti-aliased rendering. The canvas is cleared to the background color and the cursor position is reset to the origin (0, 0).
