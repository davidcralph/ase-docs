Namespace BOOSE.Interpreter.Commands

Classes

AboutCommand

Command implementation that displays information about the BOOSE application.

AssignCommand

Command for assigning values to existing variables (used by preprocessor for "width = 2*radius" syntax).

MethodBodyCommand

Internal command for storing method body lines.

MethodDefinition

Represents a method definition with parameters and body.

UnrestrictedArrayCommand

Command for declaring arrays of integers or reals. Syntax: array int arrayName size or array real arrayName size

UnrestrictedBooleanCommand

Command for declaring and initializing boolean variables. Supports both declaration only (boolean x) and declaration with initialization (boolean x = true).

UnrestrictedCallCommand

Command for calling a method. Syntax: call methodName arg1 arg2 ...

UnrestrictedCircleCommand

Unrestricted circle command that accepts any radius size.

UnrestrictedEndMethodCommand

Command for ending a method definition. Syntax: end method

UnrestrictedIntCommand

Command for declaring and initializing integer variables. Supports both declaration only (int x) and declaration with initialization (int x = 5).

UnrestrictedMethodCommand
UnrestrictedMoveToCommand

Unrestricted moveto command that accepts any coordinates.

UnrestrictedPeekCommand

Command for reading a value from a specific index in an array into a variable. Syntax: peek varName = arrayName index

UnrestrictedPenCommand

Unrestricted pen command that accepts variable references for color values.

UnrestrictedPokeCommand

Command for setting a value at a specific index in an array. Syntax: poke arrayName index = value

UnrestrictedRealCommand

Command for declaring and initializing real (double) variables. Supports both declaration only (real x) and declaration with initialization (real x = 5.5).

UnrestrictedRectCommand

Unrestricted rectangle command that accepts any width and height.

UnrestrictedTriCommand

Unrestricted triangle command that accepts any width and height.

UnrestrictedWriteCommand

Command implementation that writes text or evaluates expressions and displays the result.

VariableAssignmentCommand

Command for assigning values to existing variables. Handles expressions like "width = cheese*2" or "x = 10".
